Open-source cookie consent and Google Consent Mode plugin for WordPress.

== Description ==

Yablonsky Cookie Consent for WordPress is a self-hosted cookie consent banner and Google Consent Mode v2 integration plugin for WordPress websites.

This package currently contains the Phase 1 development skeleton. It includes the plugin bootstrap, admin settings page, settings model, sanitization helpers, and public documentation structure.

This plugin is designed for WordPress websites that need a self-hosted cookie banner, granular consent, and Google Consent Mode v2 support for Google Analytics, Google Tag Manager, and Google Ads conversion tracking.

It is not a Google-certified CMP and does not implement IAB TCF for Google publisher products such as AdSense, Ad Manager, or AdMob.

== Installation ==

1. Upload the plugin ZIP through WordPress admin.
2. Activate the plugin.
3. Go to Settings > Yablonsky Cookie Consent.
4. Configure the basic settings.

== Frequently Asked Questions ==

= Does this plugin guarantee Google Ads approval? =

No. This plugin helps make a WordPress website Google Consent Mode v2 ready and supports cleaner Google Analytics, Google Tag Manager, and Google Ads conversion tracking setups. It cannot guarantee Google Ads account, campaign, or landing page approval.

= Is this a Google-certified CMP? =

No. This plugin is not a Google-certified CMP and does not implement IAB TCF.

= Does this plugin phone home? =

No. The plugin does not send usage telemetry to Yablonsky.io.
